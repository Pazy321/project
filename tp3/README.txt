кароче для запуска надо создать в MYsql БД (Код брать из файла SQl.txt) альше мы переходим в файл .env и меняем там 

DB_PASSWORD=your_mysql_password_here - пароль
DB_NAME=glampling - название бд

npm install

npm run init-db

npm start
